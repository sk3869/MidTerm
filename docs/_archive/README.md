# Archive

This folder holds closed workstreams and completed research.

## Structure

```
_archive/
  specs/    <- completed workstream folders moved here after all quality gates pass
  research/ <- research memos that have been fully consumed into specs
```

## When to Archive

Move a workstream to `_archive/specs/` when:
1. All acceptance criteria are met
2. All quality gates pass
3. The work is merged to main

Move a research memo to `_archive/research/` when:
1. The research has been fully incorporated into a sprint doc
2. No further updates are expected
