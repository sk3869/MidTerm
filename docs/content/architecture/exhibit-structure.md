# Exhibit Structure

## Standard Exhibit Page Architecture

Every exhibit page follows this structure:

```
ExhibitHeader
  - label (e.g. "Exhibit 01")
  - title (e.g. "Personal Computers")
  - eraRange (e.g. "1970s – 1990s")
  - overview (2 paragraphs of historical context)

ArtifactGrid
  - 3 ArtifactCards

CulturalImpact
  - title
  - 2-3 paragraphs analyzing cultural significance

SourcesPanel
  - 2+ authoritative sources with institution name and URL
```
